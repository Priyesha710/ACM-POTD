**Description:**

I first did a basic delete elements while going through the list. That took O(n^2) time as deleting at max can be n elements for each element of the vector.
So next, I tried a different approach, I appended each unique element (found using two pointers, as the list is sorted, each unique element will come after the previous one) at the beginning of the vector. This took O(n) time. Next, I resized my vector to include only the first unique elements. This too took O(n) time.

Overall time complexity: O(n).

Overall space complexity: O(1) (in-place solution).
